Usage Example: distant_point_on_circle
Function docs: https://splashkit.io/api/geometry/#distant-point-on-circle

Minimal program that draws a grey circle, a red source point, and the blue farthest point on the circle from that source. One frame, then exit.

Why useful: shows exactly what the function returns, with no loops or extra scaffolding.

How to run:
- C++:  skm clang minimal_distant_point_on_circle.cpp
- C#:   build & run either C# file with dotnet
- Python: run with SplashKit Python bindings installed

Files: C++, C# (top-level + OOP), Python, this text file, and a screenshot.
