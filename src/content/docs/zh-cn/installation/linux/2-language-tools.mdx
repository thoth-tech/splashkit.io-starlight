SplashKit 支持多种编程语言，每种语言都有自己的工具集。请按照以下说明安装您选择的语言所需的工具。

## 特定语言安装

### C# 工具

要在 SplashKit 中使用 C# 进行开发，您需要安装 .NET：

<Tabs syncKey="linux-distro">
<TabItem label="Ubuntu/Debian">

<LinkButton icon="external" href="https://learn.microsoft.com/en-us/dotnet/core/install/linux-ubuntu" target="_blank">在 Ubuntu 上安装 .NET SDK 或 .NET 运行时</LinkButton>
<LinkButton icon="external" href="https://learn.microsoft.com/en-us/dotnet/core/install/linux-debian" target="_blank">在 Debian 上安装 .NET SDK 或 .NET 运行时</LinkButton>

</TabItem>
<TabItem label="Arch">

在 Arch 上安装 .NET SDK：

```shell
yay -S dotnet-sdk
```

仅安装 .NET 运行时（用于运行 .NET 应用程序而无需开发工具）：

```shell
yay -S dotnet-runtime
```

</TabItem>
<TabItem label="Fedora">

<LinkButton icon="external" href="https://learn.microsoft.com/en-us/dotnet/core/install/linux-fedora" target="_blank">在 Fedora 上安装 .NET SDK 或 .NET 运行时</LinkButton>

</TabItem>
</Tabs>

如果 .NET 安装程序没有自动更新您的 shell 配置文件，您可以按照[找不到 'dotnet' 命令](/zh-cn/troubleshoot/linux/issue-6-dotnet-not-found)故障排除页面上的步骤手动添加它。

### C++

对于 C++，如果您有标准的 Linux 开发环境，您应该已经拥有必要的工具。SplashKit 无需额外的特殊设置。

### Python

大多数 Linux 发行版都预装了 Python。如果没有安装 Python，或者您需要额外的 Python 工具，请按照以下步骤操作。

<Steps>

1. **安装 Python**

    根据您的 Linux 发行版使用以下命令安装 Python 3 和 `pip`（Python 包安装程序）：

    <Tabs syncKey="linux-distro">
    <TabItem label="Ubuntu/Debian">

    ```shell
    sudo apt install python3 python3-pip
    ```

    </TabItem>

    <TabItem label="Arch">
    <Tabs syncKey="package-manager">
    <TabItem label="使用 pacman">

    ```shell
    sudo pacman -S python python-pip
    ```

    </TabItem>
    <TabItem label="使用 yay">

    ```shell
    yay -S python python-pip
    ```

    </TabItem>
    </Tabs>
    </TabItem>

    <TabItem label="Fedora">

    ```shell
    sudo dnf install python3 python3-pip
    ```

    </TabItem>
    </Tabs>

2. **验证安装（可选）**

    要确认 Python 和 `pip` 安装成功，请检查版本：

    ```shell
    python3 --version
    pip3 --version
    ```

    如果安装成功，这两个命令都应该输出版本信息。

</Steps>
