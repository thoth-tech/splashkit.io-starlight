:::tip[Using the Programmer's Field Guide?]

If you are using the Field Guide, we *strongly recommend* that you follow its installation guide:

- [Set up my Linux Box](https://programmers.guide/book/appendix/0-installation/2-0-setup-linux/)
- [Set up my Raspberry Pi](https://programmers.guide/book/appendix/0-installation/2-3-setup-pi/#3-setup-raspberry-pi-os) (if using a Raspberry Pi)

These guides also include an automated script for an easier installation option.

:::
