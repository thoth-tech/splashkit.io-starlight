:::caution[Issue:]

You receive an error when attempting to install SplashKit using `skm global install`, or when compiling SplashKit code, itâ€™s likely because the system cannot locate the necessary SplashKit library files. Adding these paths manually can resolve the issue.

![Error Example](/images/installation/linux/skm-global-linux-fail.png)

:::

## Solution(s)

### 1. Update Your Shell Profile (`~/.bashrc` or `~/.zshrc`) using echo

Follow these steps to update your shell profiles and system library path to include SplashKit.

**For Bash Users**:

<Tabs syncKey="shell-profile">
<TabItem label="Bash Users">

<Steps>

1. To add SplashKit to your PATH, run the following command:

    ```zsh
    echo "export PATH=\"$HOME/.splashkit:\$PATH\"" >> ~/.bashrc
    ```

2. **Apply Changes**

    Reload the `.bashrc` paths with the following command to apply changes immediately:

     ```shell
     source ~/.bashrc
     ```

3. Run the global install command again using:

    ```shell
    skm global install
    ```

</Steps>

</TabItem>
<TabItem label="Zsh Users">

<Steps>

1. To add SplashKit to your PATH, run the following command:

    ```zsh
    echo "export PATH=\"$HOME/.splashkit:\$PATH\"" >> ~/.zshrc
    ```

2. **Apply Changes**

    Reload the `.zshrc` paths with the following command to apply changes immediately:

     ```shell
     source ~/.zshrc
     ```

3. Run the global install command again using:

    ```shell
    skm global install
    ```

</Steps>

</TabItem>
</Tabs>

### 2. Update Your Shell Profile (`~/.bashrc` or `~/.zshrc`) using nano

Follow these steps to update your shell profiles and system library path to include SplashKit.

**For Bash Users**:

<Tabs syncKey="shell-profile">
<TabItem label="Bash Users">

<Steps>

1. Open your `~/.bashrc` file:

    ```shell
    nano ~/.bashrc
    ```

2. Add the following line near the end of the file:

    ```shell
    export PATH="$PATH:$HOME/.splashkit"
    ```

3. Save and close the file, then update your profile:

    ```shell
    source ~/.bashrc
    ```

4. Run the global install command again using:

    ```shell
    skm global install
    ```

</Steps>

</TabItem>
<TabItem label="Zsh Users">

<Steps>

1. Open your `~/.zshrc` file:

    ```shell
    nano ~/.zshrc
    ```

2. Add the following line near the end of the file:

    ```shell
    export PATH="$PATH:$HOME/.splashkit"
    ```

3. Save and close the file, then update your profile:

    ```shell
    source ~/.zshrc
    ```

4. Run the global install command again using:

    ```shell
    skm global install
    ```

</Steps>

</TabItem>
