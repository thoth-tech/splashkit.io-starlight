The SplashKit SDK is regularly updated to provide compatibility with the latest features, enhancements, and bug fixes. If you encounter issues or notice differences between your SDK and the latest documentation, updating SplashKit may resolve these.

## How to update SplashKit

To update SplashKit on your system, use the following command in your terminal:

```shell
skm update
```

This command fetches the latest version of SplashKit and updates your SDK to the most recent release.
