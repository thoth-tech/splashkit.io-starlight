# SplashKit 故障排除

欢迎来到 SplashKit 故障排除指南！本页面将帮助您解决在安装和使用 SplashKit 过程中可能遇到的常见问题。

## 常见问题分类

### Windows 相关问题
- **MSYS2**: 在 Windows MSYS2 环境中的常见问题
- **WSL**: 在 Windows 子系统 (WSL) 中的常见问题

### macOS 相关问题
- 特定于 macOS 系统的安装和运行问题

### Linux 相关问题
- 各种 Linux 发行版的常见问题

## 获取帮助

如果您在这里找不到解决方案：

1. 查看我们的 [GitHub Issues](https://github.com/splashkit/splashkit-core/issues)
2. 在我们的社区论坛寻求帮助
3. 提交新的问题报告
