Create Animation (index + sound) — create an Animation from a script using index and optionally play starting-frame audio

This example demonstrates `create_animation(script, idx, with_sound)` — create a brand new `Animation` from an `AnimationScript` by integer index and optionally play audio attached to the starting frame.
