# 在 Windows 子系统 (WSL) 中安装 SplashKit

本指南将帮助您在 Windows 子系统 Linux (WSL) 环境中安装 SplashKit。

## 前提条件

- Windows 10 版本 1903 或更高版本
- 已启用 WSL 功能
- 已安装 Ubuntu 或其他 Linux 发行版

## 安装步骤

### 1. 启用 WSL

如果尚未启用 WSL，请在 PowerShell 中运行：

```powershell
wsl --install
```

### 2. 更新系统

在 WSL 终端中：

```bash
sudo apt update && sudo apt upgrade
```

### 3. 安装依赖项

```bash
sudo apt install build-essential curl git
```

### 4. 安装 SplashKit

使用我们的一键安装脚本：

```bash
bash <(curl -s https://raw.githubusercontent.com/splashkit/skm/master/install-scripts/skm-install.sh)
```

## 图形界面配置

WSL 中运行图形应用可能需要额外配置。
