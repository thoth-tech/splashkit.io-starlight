## 更新 SplashKit

SplashKit SDK 定期更新，以提供与最新功能、增强功能和错误修复的兼容性。如果您遇到问题或注意到您的 SDK 与最新文档之间存在差异，更新 SplashKit 可能会解决这些问题。

---

要将系统上的 SplashKit 更新到最新版本，请在终端中使用以下命令：

```shell
skm update
```
