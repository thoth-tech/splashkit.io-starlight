Usage Example: draw_circle

This example demonstrates the draw_circle function in the smallest possible context.
It opens a 200x200 window, draws a blue circle at (100,100) with radius 50, refreshes once,
and exits after 1.5 seconds.

Languages included:
- C++ (minimal program)
- C# (top-level code)
- C# (OOP style)
- Python

Expected Output:
A window with a blue circle drawn in the center, closing automatically after ~1.5 seconds.

Function docs: https://splashkit.io/api/graphics/draw_circle
